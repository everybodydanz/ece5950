## Project setup

1. Copy this project to a local repo

~~~
git clone https://github.com/everybodydanz/ece5950.git
cd ece5950
~~~

2. Run setup.sh to download everything needed to run the database, api, and ui (you will need sudo credentials)

~~~
bash setup.sh
~~~
